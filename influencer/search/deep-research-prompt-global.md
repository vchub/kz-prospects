You are a senior market intelligence analyst for creator economy / influencer-tech products.

Research goal:
Find and analyze successful and unsuccessful influencer marketing / creator-commerce platforms similar to this concept:
- brand offer marketplace
- creator workflow (acceptance, briefing, delivery, approvals)
- hybrid monetization (fixed + performance: CPA/CPS/CPC/CPV)
- attribution layer (links, promo codes, pre-landers, marketplace/on-site content)
- contracts, rights management, payouts, compliance/tax support
- multi-channel support (TikTok, Instagram, YouTube, Telegram, Twitch, X, Facebook, Snapchat, marketplaces)

Scope:
- Geography: global (US, EU, LATAM, MENA, APAC, CIS)
- Time window: 2018 to present (include latest available info)
- Include both winners and failures (shutdowns, pivots, stagnation, down-rounds)

Required outputs:
1) Comparable projects dataset (minimum 30 platforms)
- 15+ successful/scaled
- 10+ unsuccessful/failed/pivoted
- 5+ emerging challengers (founded recently or major relaunch)

2) For each platform, extract:
- Company/platform name, launch year, HQ, markets
- Primary channels supported
- Core product model
- Key features (workflow, attribution, payments, compliance, AI, rights/licensing, analytics)
- Monetization/pricing model
- Target segment (SMB, enterprise, D2C, agencies, creators tier)
- Proof of traction (revenue, GMV, customers, creator count, retention proxies, funding, profitability signals)
- Critical events timeline (funding, acquisitions, layoffs, pivots, shutdown risk)
- Why it succeeded/failed (with evidence)
- Current status (active, acquired, sunset, pivoted)

3) Feature benchmarking:
- Build a feature matrix across all platforms.
- Mark each feature as: "table stakes", "differentiator", or "declining value".
- Highlight which feature combinations correlate with better outcomes.

4) Trends analysis (2024–2026 emphasis):
- Product trends
- Business model trends
- Measurement/attribution trends
- AI workflow trends
- Commerce integration trends
- Regulatory/compliance trends
For each trend: evidence, maturity stage, winners benefiting, losers exposed, likely 12–24 month direction.

5) Strategic conclusions:
- Top 10 most promising features for a new entrant now
- Top 10 failure patterns to avoid
- 3 viable product-positioning options (with tradeoffs)
- "What to build first" recommendation at concept level (not execution plan)

Research quality constraints:
- Use primary/authoritative sources first: official company docs, investor reports, regulatory filings, product documentation, earnings calls, credible investigative media.
- Avoid generic blogs/listicles unless used only as secondary context.
- Every major claim must include a citation URL and date.
- Distinguish facts vs inference explicitly.
- If data is missing, state "no reliable public data found".

Output format:
A) Executive summary (1 page)
B) Table 1: Successful platforms
C) Table 2: Unsuccessful/pivoted platforms
D) Table 3: Feature matrix
E) Table 4: Trend radar (Now / Next / Watchlist)
F) Evidence-backed insights and recommendations
G) Source list (grouped by platform and trend)

Scoring framework:
- Success Score (0–100): growth, monetization durability, retention signals, defensibility
- Failure Risk Score (0–100): attribution fragility, dependency risk, unit economics risk, compliance/payment risk
Show scoring logic clearly.

Important:
Be critical. Do not just describe features; evaluate causal impact.
Flag contradictions across sources.
Prioritize evidence from the most recent available dates.
